Aproveitando ainda mais o youKnow

Após mostrar como utilizar tipografias diferentes no youKnow, mostro nessa publicação como inserir imagens.

O youKnow não possui um botão para inserir imagem

As etapas a seguir consideram que você possui a imagem em seu computador. Se você quer utilizar uma imagem que já está na Internet, pule para o passo 2.

Passo 1: Coloque a imagem na internet

Para aplicar a imagem, antes precisamos subí-la para algum servidor ou serviço. Há diversos serviços de hospedagem de imagem. Experimente e veja qual você prefere.

Usarei o Imgur para essa publicação.

Acesse o site doImgur e clique no botão "Upload Images".

Clique em "Browse your computer" para buscar a imagem em seu computador.

Nesse exemplo, não irei compartilhar essa imagem com a comunidade e como somente subirei uma imagem, não tenho como criar um álbum. Simplesmente clico em "Start Upload".

O que precisamos do Imgur é o endereço final dela. Você pode achar em "Direct Link", dentro da caixa "Share this Image". Você pode clicar no ícone ao lado do endereço para copiá-lo.

De posse desse endereço, podemos ir para o youKnow.

Passo 2: Código-font FTW!

Depois da minha última publicação, você já sabe como acessar o código fonte daquilo que está escrevendo... certo? Mais uma vez, acesse-o.

Infelizmente, apenas colar aquele endereço que copiamos no passo 1 (ou o endereço de sua imagem se você veio direto para o passo 2) no código fonte não fará funcionar.

Para a imagem aparecer como imagem, precisamos aplicar o endereço em uma tag HTML própria para isso: a tag IMG. O código para inserir a imagem deve ser...

<img src="http://i.imgur.com/AXbAhn7.png" >

...apenas trocando o endereço dentro de "src" para o endereço de sua imagem. Na sua publicação deve ficar assim:

Quando voltar para a visualização da sua publicação, lá estará sua imagem aplicada.

A imagem nesse exemplo ficou um pouco grande, não acha? Eu acho. Por esse motivo, vou acrescentar algumas poucas linhas de código para ela aparecer mais comportada.

<style>   .minha-publicacao img { display: block; height: auto; margin-left: auto; margin-right: auto; max-width: 80%; }   </style>   <div class="minha-publicacao"> <p> ...mas como?</p> <img src="http://i.imgur.com/AXbAhn7.png" > </div>

Pronto! O que escrevemos foi: todas as imagens dentro de "minha-publicao" deverão ter no máximo 80% do tamanho e deverão estar centralizadas.

Conclusão

Vimos como inserir imagens no youKnow utilizando serviços de hospedagem de imagens como o Imgur e de quebra as aplicamos de uma forma que não extrapolarão a largura de sua publicação.

Serviu também para ter mais um pouco de contato com o código fonte de sua publicação, onde podemos alterar livremente sua aparência com algumas linhas de código.
